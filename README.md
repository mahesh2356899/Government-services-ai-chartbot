# Government Services AI Chatbot

## 📌 Project Description

Government Services AI Chatbot is a web-based application designed to help citizens easily access information about various government services.
The chatbot allows users to ask questions through text or voice and receive instant responses about services like Aadhaar, PAN, ration card, pensions, and other government schemes.

## 🚀 Features

* AI-based chatbot for answering government service queries
* Voice input using microphone
* Speaker output for chatbot responses
* Simple and user-friendly interface
* Card-based service display
* Quick access to common government services

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Web Speech API (for voice recognition and speech output)

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder in **VS Code**.
3. Open the `index.html` file.
4. Click **Run** or open with **Live Server**.
5. The chatbot will open in your browser.

## 💡 Example Queries

Users can ask questions like:

* "How to apply for Aadhaar card?"
* "How to get a ration card?"
* "What are the documents required for PAN card?"
* "Government schemes for farmers"

## 🔊 Voice Features

* Click the **mic button** to speak your question.
* The chatbot will recognize the speech and respond.
* The **speaker feature** will read the answer aloud.
